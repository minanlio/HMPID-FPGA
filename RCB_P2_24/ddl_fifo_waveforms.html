Sample behavioral waveforms for design file ddl_fifo.tdf

The following waveforms show the behavior of scfifo megafunction for the chosen set of parameters in design ddl_fifo.tdf. The design ddl_fifo.tdf has a depth of 1024 words of 32 bits each. The output of the fifo is registered. The fifo is in legacy synchronous mode. The data becomes available after 'rdreq' is asserted; 'rdreq' acts as a read request.

Fig. 1 : Wave showing read and write operation.
